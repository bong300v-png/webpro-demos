## Steps

1. Stage all changes
```bash
git add -A
```

2. Check status
```bash
git status --short
```

3. Commit with descriptive message (replace MESSAGE with context)
```bash
git commit -m "MESSAGE"
```

4. Push to origin master
```bash
git push origin master
```
